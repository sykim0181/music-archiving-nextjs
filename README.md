## 프로젝트 소개
좋아하는 앨범을 아카이브할 수 있는 서비스입니다.

Spotify Web API를 활용해 앨범을 저장하고 재생할 수 있습니다.

**링크**: [https://music-archiving.vercel.app/](https://music-archiving.vercel.app/)   


## 기술 스택
- React
- Next.js
- Typescript
- Redux
- Supabase


## 주요 기능
- 앨범 컬렉션 저장
  - 앨범 검색 및 추가
  - 공개 저장 가능 (포스트)
- 타 사용자의 컬렉션 조회
- 인터랙티브한 앨범 재생
  - 마우스로 LP를 움직여 재생
